什么是 jQuery

jQuery 是 js 的一个库，封装了我们开发过程中常用的一些功能，方便我们调用，提高开发效率。

js库是把我们常用的功能放到一个单独的文件中，我们用的时候，直接引用到页面里即可。

jQuery 的两大特点

（1）链式编程：比如.show()和.html()可以连写成.show().html()。节省代码量,编程更优雅

链式编程原理：return this。

通常情况下，只有设置操作才能把链式编程延续下去。因为获取操作的时候，会返回获取到的相应的值，无法返回 this。

（2）隐式迭代：隐式 对应的是 显式。遍历内部DOM元素（伪数组形式存储）的过程就叫做隐式迭代, 简单理解：给匹配到的所有元素进行循环遍历，执行相应的方法，而不用我们再进行循环，简化我们的操作，方便我们调用。

如果获取的是多元素的值，大部分情况下返回的是第一个元素的值。

jq顶级对象

  1. $是 jQuery的别称(另外的名字)，在代码中可以使用 jQuery代替$，但一般为了方便，通常都直接使用$.

参数可以省略, 省略后无动画,直接显示

参数详情:
speed: 三种预定速度之一的字符串: slow(慢), normal(正常), fast(快),或者表示动画时长的毫秒数值
easing: 用来指定切换效果, 默认 swing(摇摆) ,可用参数 linear(线性)
fn: 回调函数,在动画完成时执行的函数,每个元素执行一次

对象拷贝

obj里面的内容拷贝到targetObj, targetObj原来有的数据会被覆盖掉

  1. 浅拷贝(为空或者false)把原来对象里面的复杂数据类型地址拷贝给目标对象
    修改了拷贝后对象的值,原来对象的值也会跟着改变
    $.extend(targetObj, obj);

  2. 深拷贝(前面加true)把里面的数据完全复制一份给目标对象 如果里面有不冲突的属性,会合并到一起
    $.extend(true, targetObj, obj);

修改了拷贝后对象的值,不会影响原来对象的值

多库共存

jQuery使用$作为标示符，随着 jQuery的流行其他js库也会用这$作为标识符，这样起使用会引起冲突. 需要一个解决方案，让 jQuery和其他的js库不存在冲突，可以同时存在，这就叫做多库共存。

解决方案:

  1. 把里面的$符号统改为 jQuery.比如 jQuery("div")
  2. jQuery变量规定新的名称：$.noConflict() 比如 var x = $.noConflict()
