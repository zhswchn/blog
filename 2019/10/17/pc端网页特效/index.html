pc端网页特效

本文最后更新于：2021年3月7日 晚上

本篇为pc端网页特效笔记,记录下来主要用于复习,巩固。包括以下内容：三大系列、鼠标经过的区别、匀速动画、缓速动画、动画函数的封装、多个值之间移动、节流阀。

offset系列

offset的中文是：偏移，补偿，位移。

js中有一套方便的获取元素尺寸的办法就是offset家族。offset家族包括：

  • offsetWidth

  • offsetHight

  • offsetLeft

  • offsetTop

  • offsetParent

offsetWidth: 得到元素的大小,宽度 是包含padding + border + width的
offsetHight:得到元素的大小,高度 是包含padding + border + height的

这两个属性，他们绑定在了所有的节点元素上。获取之后，只要调用这两个属性，我们就能够获取元素节点的宽和高。

offsetLeft:

返回距离上级盒子（带有定位）如果没有父元素,或者父元素没有定位, 则以 body 为准, (数值不带单位), 从父亲的 padding 开始算，父亲的 border 不算。(数值不带单位)
offsetTop: 可以得到元素的偏移位置的数值

offsetParent:

检测父系盒子中带有定位的父盒子节点。返回结果是该对象的父级（带有定位）。

  • 如果当前元素的父级元素，没有CSS定位（position为absolute、relative、fixed），那么offsetParent的返回结果为body。

  • 如果当前元素的父级元素，有CSS定位（position为absolute、relative、fixed），那么offsetParent的返回结果为最近的那个父级元素。

parentNode: 返回的是最近一级的父级元素,不管父级元素带不带定位

offset与style的区别

offset:
可以得到任意样式表中的样式值,且没有单位
包含padding border width
只读,不能赋值

style:
只能得到行类样式表中的样式值,带有单位
不包含padding和border的值
可读写

一般的做法是：用offsetLeft 和 offsetTop 获取值，用style.left 和 style.top 赋值（比较方便）

  • style.left：只能获取行内式，获取的值可能为空，容易出现NaN。

  • offsetLeft：获取值特别方便，而且是现成的number，方便计算。它是只读的，不能赋值。

client系列

  1. clientWidth 和 clientHeight

和offsetWidth最大区别就是不包含边框
包含padding以及内容,不包含边框宽度,数值不带单位

盒子调用时：

  • clientWidth：获取盒子区域宽度（padding + width）。

  • clientHeight：获取盒子区域高度（padding + height）。

body/html调用时：

  • clientWidth：获取网页可视区域宽度。

  • clientHeight：获取网页可视区域高度。

clientX 和 clientY

event调用：

  • clientX：鼠标距离可视区域左侧距离。

  • clientY：鼠标距离可视区域上侧距离。

  1. clientTop 和 clientLeft

  • clientTop：盒子的上border。

  • clientLeft：盒子的左border。

scroll系列

  1. ScrollWidth 和scrollHeight

获取盒子的宽高。调用者为节点元素。不包括 border和margin。如下：

  • scrollWidth = width + padding;

  • scrollHeight = height + padding;

scrollHeight有一个特点：如果文字超出了盒子，高度为内容的高（包括超出的内容）；不超出，则是盒子本身高度。

  1. scrollTop 和 scrollLeft

网页被卷去的头部和左边的部分。

比如说，一个网页往上滚动的时候，上面的部分自然被浏览器遮挡了，遮挡的高度就是scrollTop。

三大系列总结

  1. offset系列用于获取元素位置

offsetWidth: 返回自身包括padding, 边框, 内容区的宽度, 不带单位

  • offsetWidth = width + padding + border
  1. client系列用于获取元素大小

clientWidth: 返回自身包括padding, 内容区的宽度, 不含边框, 不带单位

  • clientWidth = width + padding
  1. scroll系列用于获取滚动距离, 页面的滚动距离用window.pageXOffset获得

scrollWidth: 返回自身实际的宽度, 不含边框, 返回值不带单位

  • scrollWidth = 内容宽度（不包含border）

mouseover和mouseenter的区别(鼠标经过)

mouseover鼠标经过自身盒子会触发,经过子盒子还会触发

mouseenter只会经过自身盒子触发,不会冒泡,搭配mouseleave鼠标离开 ,也不会冒泡

匀速动画

动画原理:

  1. 获得盒子当前位置
  2. 让盒子在当前位置加上1个移动距离
  3. 利用定时器不断重复这个操作
  4. 加一个结束定时器的条件
  5. 注意此元素需要添加定位， 才能使用 element.style.left

动画函数封装

简单动画函数封装 obj: 目标对象, target: 目标位置

缺点: 性能不好,每次调用,var就会生成新的timer空间

缓动动画

思路：

  1. 让盒子每次移动的距离慢慢变小， 速度就会慢慢落下来。
  2. 核心算法：(目标值 - 现在的位置) / 10 做为每次移动的距离 步长
  3. 停止的条件是： 让当前盒子位置等于目标位置就停止定时器

匀速动画就是 盒子是当前的位置 + 固定的值
缓动动画就是 盒子当前的位置 + 变化的值(目标值 - 现在的位置) / 10）
